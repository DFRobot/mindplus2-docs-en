# 3.1 实时模式
实时模式是Mind+中的一种基础编程方式，它允许用户通过积木编程实时控制硬件或舞台角色的行为。程序在电脑上运行时，指令会即时传输到设备或舞台，实现互动式控制。这种模式适合没有编程基础的用户，或者想要快速验证想法、制作互动项目的人群。

**特点：**

**指令执行即时响应，操作直观。**

无需上传程序到硬件，可随时调整和测试。

适合初学者和快速原型制作。
