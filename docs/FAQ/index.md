# 常见问题

本页面收集了Mind+软件使用过程中的常见问题和解决方案。

## 安装相关问题

### 问题1：Mind+软件无法正常安装
**解决方案：** 请确保您的系统满足最低配置要求，并以管理员权限运行安装程序。

### 问题2：安装完成后无法启动
**解决方案：** 检查防火墙设置，确保Mind+被允许通过防火墙。

## 使用相关问题

### 问题3：无法连接硬件设备
**解决方案：** 
1. 确认设备驱动已正确安装
2. 检查USB连接线是否正常
3. 重新启动Mind+软件

### 问题4：代码上传失败
**解决方案：** 检查端口设置是否正确，确保设备处于正确的连接状态。
